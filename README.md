This project allows you to download sounds from YouTube using files in .csv(Artist,Track) or .txt(Artist - Track) format.
The main code only works with .csv files of the specified format, namely Artist and Track. This project also includes a converter from .txt (Artist - Track format) to .csv .
This project also allows you to combine two .csv files (Artist,Track) and eliminate duplicate titles.
In order to load sounds, you need to select the appropriate item and have a .csv file called: "final_file.csv" (! It is very important that the final file is named exactly like this!) in the same folder as the project itself, that is, in the folder with main.exe (main.py).
To run, download the dist file and run main.exe. The source code and libraries used can be found in the corresponding file.
